# NF1-Smart-Redirector-Model
AlphaFold 3 ve HADDOCK kullanarak tasarlanmış NF1-KRAS etkileşim modellemesi
---
## 🎯 Yapay Zeka Laboratuvar Doğrulaması (Mayıs 2026)
Bu projenin fikir aşaması, **AlphaFold 3** süper bilgisayar simülasyonları ve yerel Python geometrik analiz araçları ile test edilmiştir. Tasarlanan *Akilli_Saptirici_miRNA* molekülünün, kontrolden çıkan KRAS onkoprotein yolağına **2.85 Å** gibi kararlı bir hidrojen bağı mesafesiyle kusursuz şekilde kilitlendiği atomik boyutta doğrulanmıştır. Projenin teorik ilk fazı %100 başarıyla tamamlanmıştır.
